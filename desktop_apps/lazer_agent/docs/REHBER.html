AI Scriptini Bağlama

ai_module.py dosyasını bir metin editörüyle açın ve şu satırı bulun:

AGENT_SCRIPT_PATH = ""   # ← buraya scriptinizin tam yolunu yazın

Örnek:

AGENT_SCRIPT_PATH = r"C:\AtolYe\agent.py"
ℹ️
Demo Mod: Script bağlanmadan önce program mesajları echo eder ve "AI agent henüz bağlanmadı" notunu gösterir. Atölye işleri için DXF/SVG sekmesini kullanabilirsiniz.

7 🏗️ EXE Olarak Derleme

Programı internetsiz herhangi bir Windows bilgisayarda çalışacak tek bir .exe dosyasına dönüştürmek için:

cd c:\Users\DDSOUND\Desktop\exemiz\dd1_workshop_tools

dd1_workshop_tools\

İlk derleme 3–5 dakika sürebilir. Bu normaldir. Oluşan .exe dosyası Python gerektirmez, herhangi bir Windows 10/11 bilgisayarda çalışır.

8 ❓ Sık Sorulan Sorular

Görsel yükledim ama hiçbir şey çıkmıyor?

Threshold değerini düşürün (128'in altına). Görseliniz açık renkliyse Lazer Modunu açın ve tekrar dönüştürün.

DXF CorelDRAW'da açılmıyor?

Dosya → Aç değil, Dosya → İçe Aktar (Import) kullanın. Format R2010 DXF seçin.

İşlem çok uzun sürüyor?

4096×4096 üzeri görseller otomatik küçültülür. Çok büyük dosyalar için görseli önceden boyutlandırın.

AI sohbet cevap vermiyor?

ai_module.py içindeki AGENT_SCRIPT_PATH boşsa Demo Mod aktiftir. Gerçek scripti bağlamak için bu değişkeni güncelleyin.

ModuleNotFoundError hatası alıyorum?

pip install -r requirements.txt
