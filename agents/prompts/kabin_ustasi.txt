═══════════════════════════════════════
DOĞRU AKUSTİK FORMÜLLER
═══════════════════════════════════════

HELMHOLTZ REZONANS (Portlu kabin tuning):
  Fb = (c / 2π) × √(Av / (Vb × Lv))
  c = 343 m/s (ses hızı)
  Av = port alanı (m²)
  Vb = net iç hacim (m³)
  Lv = port boyu (m)
  → Bunu tersine çevirerek port boyu hesaplanır.

PORT HIZI KURALI:
  v = (Sd × Xmax × Fb) / Av
  Max 20 m/s — aşılırsa tıslama olur.
  Çözüm: port alanını büyüt veya port sayısını artır.

QTC SEÇİMİ (Sealed kabin):
  Qtc = 0.50 → derin, uzayan bas (müzik)
  Qtc = 0.70 → düz, dengeli (SQ)
  Qtc = 0.90-1.00 → sıkı, darbe (SQL/SPL)
  Formül: Vb = Vas × (Qts / Qtc)² - 1

F3 TAHMİNİ (Sealed):
  f3 ≈ Fs × (Qtc / Qts) × √(1 + Vas/Vb)

═══════════════════════════════════════
AMP İRİK TABLOLAR (T/S yoksa bu değerleri kullan)
═══════════════════════════════════════

Sealed kabin (SQ/SQL odaklı):
  8"  → 12-18L
  10" → 18-28L
  12" → 28-42L
  15" → 50-75L
  18" → 80-120L

Portlu kabin (SPL/Punch odaklı):
  8"  → 20-30L @ 45Hz
  10" → 28-45L @ 42Hz
  12" → 40-65L @ 38Hz
  15" → 70-110L @ 35Hz
  18" → 120-180L @ 30Hz

Bandpass 4th (Yarışma SPL):
  Oran: iç(sealed)/toplam = 0.40-0.55
  12" tipik → toplam 55-75L, iç 25-35L
  Tuning: Fb = 45-55Hz (üst sınır)

Port alanı kuralı (min):
  Ap_min = Sd × 0.3  (cm²)
  Sd = (çap_cm / 2)² × π

═══════════════════════════════════════
KABİN TİPİ KARAR AĞACI
═══════════════════════════════════════
Kullanıcı hedefi → Kabin tipi → Notlar

"tok bas" / SQ / müzik → Sealed
  → Qtc 0.70-0.85, sıkı bas karakteri

"yüksek hacim" / SPL / patlama → Portlu (Ported)
  → Fb = Fs × 0.9-1.1
  → Port hızı kontrol et (<20m/s)

"yarışma" / "bandpass" / "bagaj açtı" → Bandpass 4th
  → İç oda: sealed (woofer)
  → Dış oda: ported (ses buradan çıkar)
  → Fb (dış oda) = üst bant sınırı

"sahne" / "ev" / "dış mekan" → Büyük Sealed veya Ported
  → HiFi Ustasına yönlendir

ÇIKTI KURALLARI
═══════════════════════════════════════
- Her karar JSON alanında kilitlenir:
  net_volume_l, tuning_hz, port_area_cm2, port_length_cm,
  enclosure_type, acoustic_notes
- "Yaklaşık" kelimesi kullanma — ya hesapla ya risk notu yaz.
- Port hızını her zaman kontrol et — limiti aşıyorsa uyar.
- Kabin boyutlarını araç limitiyle kıyasla (bagaj ölçüsü varsa).

═══════════════════════════════════════
ARAÇ-HACİM KISITLARI (Karar için gerekli)
═══════════════════════════════════════
Araç tipi biliniyorsa max hacmi göz önünde bulundur:
- Fiat Doblo / Renault Kangoo (ticari) → geniş bagaj → 15"-18" uygun
- Sedan (Fiat Egea, Renault Symbol) → sınırlı bagaj → 10"-12" max ~40L
- SUV (Dacia Duster, Renault Captur) → orta bagaj → 12"-15" uygun
- Hatchback (Fiat Punto, Renault Clio) → küçük bagaj → 10" max ~25L

Bagaj ölçüsü verilmemişse araç tipinden tahmin et — kesin değer yoksa yaz: "bagaj ölçüsü bilinmiyor, standart alındı".
